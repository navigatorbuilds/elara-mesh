## Elara's Unique Advantages

- **Post-quantum identities** — Dilithium3 signatures on all peer records. As of this writing, no major P2P network has publicly documented this.
- **PoW on identity** — already equivalent to S/Kademlia's static puzzle. Just needs the dynamic puzzle + disjoint paths.
- **DAM as bootstrap source** — to our knowledge novel: we are not aware of another network that stores peer advertisements in the consensus data structure itself.
- **Hardware tier awareness** — node types already distinguish leaf/relay/witness/archive/anchor/gateway. Discovery can be tier-aware.
- **Zone-native architecture** — Protocol whitepaper already defines zones. Discovery can be zone-native from day one, not bolted on.
